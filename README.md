# ssh2-web
Provide web page for ssh2
为ssh2 提供一个web界面

项目的演示地址 : http://onekoko.com:4000

## 依赖于库
ssh2
=======
npm install ssh2
ssh2 https://github.com/mscdex/ssh2

express
=======
npm install express
express  http://www.expressjs.com.cn/

socket.io
=======
npm install socket.io
socket.io http://socket.io/
